# Roadmap

## Phase 1: Foundation
- CI, linting, tests, and migration strategy
- encrypted secret handling and OAuth token storage
- structured job retries, dead-letter queue, and observability

## Phase 2: Platform adapters
- YouTube Data API adapter
- Instagram/Facebook Graph API adapter
- Reddit API adapter
- X API adapter, subject to access tier and automation policy

## Phase 3: Content pipeline
- licensed source registry
- factual-source verification and citation capture
- deck renderer and template system
- approved football data provider integration

## Phase 4: Analytics
- scheduled metrics collection
- gradual content-weight optimization
- dashboards and anomaly alerts

All publishing and engagement features must use official APIs, respect rate limits, and preserve human approval controls where platform or content risk requires them.
